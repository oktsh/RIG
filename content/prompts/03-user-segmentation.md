Паттерны сегментации поведенческих данных для продуктового анализа.

## Промпт

Разработайте модель сегментации пользователей на основе поведенческих данных. Определите ключевые сегменты, их характеристики и потребности для оптимизации продукта.

## Когда использовать

- При анализе пользовательской базы для персонализации
- При планировании маркетинговых кампаний
- При определении приоритетов продуктовой разработки
- При исследовании причин оттока пользователей

## Рекомендуемые технологии

- Python -- для построения моделей кластеризации (scikit-learn, pandas)
- SQL -- для извлечения и подготовки поведенческих данных

## Пример использования

```
На основе предоставленных поведенческих данных пользователей:

1. Определите оптимальное количество сегментов (используйте метод локтя или silhouette score)
2. Опишите каждый сегмент:
   - Размер сегмента (% от общей базы)
   - Ключевые поведенческие паттерны
   - Средние метрики (LTV, retention, частота использования)
   - Потребности и болевые точки
3. Предложите стратегии для каждого сегмента:
   - Персонализация продукта
   - Коммуникационная стратегия
   - Метрики успеха
```
