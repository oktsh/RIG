Фреймворк глубокого анализа конкурентной среды с фокусом на ценностные предложения и рост.

## Промпт

Проведите комплексный анализ конкурентов: изучите их ценностные предложения, стратегии роста, позиционирование на рынке и ключевые преимущества. Определите возможности для дифференциации.

## Когда использовать

- При выходе на новый рынок или сегмент
- При пересмотре стратегии позиционирования
- При подготовке к запуску нового продукта
- При анализе угроз со стороны конкурентов

## Рекомендуемые модели

- GPT-4 -- для глубокого аналитического разбора
- Claude -- для структурированного вывода и сравнительных таблиц

## Пример использования

```
Проведите комплексный анализ конкурентов для [название продукта] в сегменте [сегмент рынка].

Для каждого конкурента определите:
1. Ценностное предложение
2. Стратегия роста
3. Ключевые преимущества
4. Слабые стороны
5. Позиционирование на рынке

Сформируйте матрицу сравнения и рекомендации по дифференциации.
```
